Lotus

2017 - Present

Lotus is a 2-D, Action-RPG that is being developed in honor of other classic 2-D adventure titles such as Metroid and Castlevania. The game was built utilizing the Unity 5 Engine with art and writing created by myself.

TurnUp Music

2016

TurnUp is a defunct music web application developed for a software development class taken at the University of Colorado Boulder. In this project, I developed the front end of the music player while my teammates handled the backend and a neural network designed to generate playlists for the project. The webapp was designed with mobile in mind so the application is reactive via CSS.
